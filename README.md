## Let's Roll!

### Clone the repo:
SSH:
```
git clone git@github.com:bsommardahl/tdd-hotel.git
```
or, Https:
```
gti clone https://github.com/bsommardahl/tdd-hotel.git
```

### Download all the dependencies:
```
cd tdd-hotel
npm install
```
