## MVP Features:

- Search Available Pets
- Reserve Pet
- Cancel Reservation
- Pet Delivery
- Pet Return

## Let's Roll!

### Clone the repo:
SSH:
```
git clone git@github.com:bsommardahl/uberpet.git
```
or, Https:
```
gti clone https://github.com/bsommardahl/uberpet.git
```

### Download all the dependencies:
```
cd uberpet
npm install
```
